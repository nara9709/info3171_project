Note

If the table width is larger than the screen, the table is scrolled horizontally. Also, if the table height is greater than 300px, it will scroll vertically. Thread tag is fixed, so it doesn't move when you scroll down.

CSS information of the table is specified in the properties of div, table, thread, tr and, th tag.

If you do not want to use both horizontal and vertical scrolling, delete the overflow-x:auto property of div style and delete the class overflow property of the same div tag as well.

Table with element

Button Checkbox URLs
Google (Absolute)
Buttons cannot be placed on a thread. Can be placed in any cell except thread. The checkbox style in the table is specified separately. ex) td input[type="checkbox"] Both relative and absolute URLs are available.

Note

Multiple checkboxes or buttons can be placed in one cell as needed. When specifying CSS, you must use inline or inline-block for the display of input elements to place more than one checkbox or button in one cell.

If you have a word you want to emphasize, you can place em tag, strong tag between the words to partially change the font thickness and inclination.

Depending on the table class, different table properties can be specified. The class of the table used is sample-table3.
