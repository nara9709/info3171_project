Charts

Line chart

Notes: Line charts show information that changes over time. 2017 - 2022.

Notes: Represent numerical values compared to each other. The length of the bar represents the value of each variable.

Notes: Represented form of our products. Each single values of 3 products.

Area chart

Notes: A form of geospatial visualization, area maps are used to show specific values set over a map of a country, state, county, or any other geographic location.

Scatter Plot

Notes: Use the multiple colors to determine the color of the points.

Pie chart

Notes: A circular chart with triangular segments that shows data as a percentage of a whole.

Donut chart

Notes: Show the relationship of parts to a whole.
